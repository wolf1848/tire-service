Наши точки на ул. Тихоокеанская

Хабаровск — две метки на карте с описанием.

Если карта не загружается, укажите JavaScript API ключ Яндекс.Карт в атрибуте apikey в подключении скрипта выше. Метки ставятся по координатам; при доступном геокодеру позиция может уточняться.

Точки в выбранном радиусе

Нажмите «Точки рядом со мной», чтобы построить список адресов внутри выбранного радиуса (по прямой на карте).
